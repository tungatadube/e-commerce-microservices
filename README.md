# E-Commerce Microservices Platform

Production-ready microservices architecture for e-commerce system.

## Tech Stack
- Spring Boot 3.x, Java 17
- PostgreSQL, Redis, RabbitMQ
- Docker, Kubernetes
- GitHub Actions CI/CD

## Services
1. Product Service - Product catalog management
2. Order Service - Order processing
3. Payment Service - Payment handling
4. User Service - Authentication & authorization

## Quick Start
```bash
docker-compose up -d
```

See individual service READMEs for details.
